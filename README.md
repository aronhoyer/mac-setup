# Simple shell script for bootstrapping a Mac for development

This is only based on my preferred tools and apps.

## Usage

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/aronhoyer/mac-setup/HEAD/setup.sh)"
```

- Git (ensure latest)
